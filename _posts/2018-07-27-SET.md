### Pengertian SET
#### Apa itu SET?

SET adalah untuk menyimpan value (unik)

* SET terbagi 3 bagian 

* set :
    - hashset 
		- treeset
		- linkedhashset
    
    
* Latihan satu 

```

import java.util.*;
public class SET1 {
    public  static void main(String[] args){
        Set<String> SetNama = new HashSet<String>();
        SetNama.add("Wulan");
        SetNama.add("icad");
        SetNama.add("ucon");
        SetNama.add("nila");

        Set<String> SetNama2 = new LinkedHashSet<String>();
        SetNama.add("Wulan");
        SetNama.add("icad");
        SetNama.add("ucon");
        SetNama.add("nila");

        System.out.println(SetNama);
        System.out.println(SetNama2);
    }

}
```
